# Episode 02 - Linux Terminal Keyboard Shortcuts

## Overview

In this episode, I learned useful keyboard shortcuts that improve productivity while working in the Linux terminal. Using these shortcuts can make navigation, multitasking, and command execution much faster.

---

# Why Learn Keyboard Shortcuts?

Using keyboard shortcuts helps to:

- Increase productivity
- Navigate the terminal efficiently
- Reduce dependence on the mouse
- Improve workflow during system administration and cybersecurity tasks

---

# Keyboard Shortcuts

| Shortcut | Description |
|----------|-------------|
| `Ctrl + Shift + T` | Open a new terminal tab |
| `Ctrl + Shift + N` | Open a new terminal window (supported in some terminals) |
| `Ctrl + L` | Clear the terminal screen |
| `Ctrl + Shift + +` | Zoom in (Increase terminal font size) |
| `Ctrl + -` | Zoom out (Decrease terminal font size) |
| `Ctrl + 0` | Reset terminal zoom to default |
| `Ctrl + Shift + C` | Copy selected text from the terminal |
| `Ctrl + Shift + V` | Paste text into the terminal |
| `Ctrl + Shift + W` | Close the current terminal tab |
| `Ctrl + Page Up` | Switch to the previous terminal tab |
| `Ctrl + Page Down` | Switch to the next terminal tab |
| `Ctrl + Shift + Q` | Quit the terminal application (may prompt for confirmation) |
| `Ctrl + ~` | Switch between multiple terminal windows (varies by desktop environment) |

---

# Most Useful Shortcuts

The shortcuts I expect to use most frequently are:

- `Ctrl + Shift + T` → Open a new terminal tab
- `Ctrl + L` → Clear the screen
- `Ctrl + Shift + C` → Copy
- `Ctrl + Shift + V` → Paste
- `Ctrl + Shift + +` → Zoom in
- `Ctrl + -` → Zoom out

---

# Notes

- Some shortcuts may vary depending on the Linux distribution or terminal emulator (GNOME Terminal, Konsole, Xfce Terminal, etc.).
- Most modern Linux terminals support these shortcuts by default.
- Learning these shortcuts saves time during penetration testing, system administration, and everyday command-line work.
